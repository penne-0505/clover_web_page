## Overview
PricingComponent で使用する表示文言は `src/constants/pricingCopy.js` に集約する。

## Keys
- `tabs`: プラン種別タブのラベル
- `intervalToggle`: 月/年切り替えと補助文言
- `planCard`: プランカードの表示ラベル
- `features`: プラン特典リスト
- `cta`: CTA ボタン文言

## Usage
- `src/components/ui/PricingComponent.jsx` で `PRICING_COPY` を参照する。
