## Overview
本文/見出し/ラベルのウェイトを整理し、ページ間の階層表現を統一する。

## Decision
- 本文は Regular、UI ラベルは Semi Bold、見出しは Bold/Black を基本とする。
- Google Fonts のウェイトを 400/600/700/900 に揃える。
- グローバルなタイポグラフィ設定を優先し、個別調整は最小限にする。

## Rationale
- 視認性と情報階層を明確化し、読みやすさを維持するため。

## Outcome
